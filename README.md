# Blog 만들기

---

### Directory Structure

- contents
  블로그 포스트 관련 파일들을 저장하기 위한 디렉토리
- src
  - components <br />
    Components를 관리하기 위한 디렉토리 입니다.
  - hooks <br />
    Custom hooks를 관리하기 위한 디렉토리 입니다.
  - pages <br />
    페이지의 역할을 하는 컴포넌트를 저장하기 위한 디렉토리 입니다.
  - templates <br />
    게시글 페이지와 같이 페이지의 역할을 하며넛 같은 형식의 여럭 콘텐츠를 보여주는 컴포넌트를 저장하기 위한 디렉토리 입니다.
